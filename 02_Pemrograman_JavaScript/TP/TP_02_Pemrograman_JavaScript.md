# Tugas Pendahuluan 02: Pemrograman JavaScript

## Hasil Awal
Input: [2, 0, 26, 28, -2]  
Output: 0  

## Analisis
Output bernilai 0 karena kondisi menggunakan `arr[i] >= 0`.
Nilai 0 ikut dikalikan sehingga hasil menjadi 0.

## Perbaikan
Mengubah kondisi menjadi `arr[i] > 0` agar hanya bilangan positif yang dihitung.
