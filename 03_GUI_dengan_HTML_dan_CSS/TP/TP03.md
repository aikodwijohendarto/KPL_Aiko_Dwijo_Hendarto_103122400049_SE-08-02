## Penjelasan

Program ini merupakan halaman web sederhana yang digunakan untuk mengubah gaya teks yang dimasukkan oleh pengguna.  
Pengguna dapat mengetikkan kalimat pada textarea, kemudian memilih aksi seperti mengubah semua huruf menjadi kapital, huruf kecil, ataupun memformat teks menjadi paragraf.

Agar tampilan halaman terlihat rapi, seluruh konten ditempatkan di tengah layar menggunakan **Flexbox** pada elemen `body`. Dengan teknik ini posisi elemen dapat disejajarkan secara horizontal dan vertikal.

Font yang digunakan adalah **Inconsolata** yang diambil dari Google Fonts sehingga tampilan teks terlihat konsisten dan lebih mudah dibaca.

Contoh pengaturan CSS yang digunakan untuk menempatkan konten di tengah halaman:

```css
body {
  font-family: 'Inconsolata', monospace;

  display: flex;
  justify-content: center;
  align-items: center;

  min-height: 100vh;
  margin: 0;
  background-color: #ffffff;
}
